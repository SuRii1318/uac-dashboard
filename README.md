# UAC System Capacity Dashboard

Interactive Streamlit dashboard for monitoring Unaccompanied Alien Children (UAC) care load, net intake, and capacity strain.

## Features
- Total system load (CBP + HHS)
- Net intake pressure (transfers minus discharges)
- Discharge offset ratio
- Strain period detection
- Date range & granularity filters
